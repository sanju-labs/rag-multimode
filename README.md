# RAG Multimodal System

A Retrieval-Augmented Generation (RAG) system that allows users to query financial reports using LLMs.

## Features
- PDF ingestion
- Vector search using FAISS
- LLM powered answers
- Modular architecture

## Tech Stack
Python
FAISS
OpenAI API
FastAPI / Streamlit

## Project Structure
app.py
rag.py
retriever.py
vectorstore.py

## Example Query
"What is Apple's total revenue in 2025?"

## Installation
pip install -r requirements.txt
python app.py
